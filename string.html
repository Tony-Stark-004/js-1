Count Characters

You are given a string S, and your task is to return an array B(having a size of 2), where B[0]contains the count of character A (uppercase) in string S and B[1]contains the count of character D (uppercase) in string S.

const countCharacters = (S) => {

    // S.sort();
    let B = [];
  
     let arr = S.split("");
     arr.sort();
  
    let count = 1;
    for(let i = 0; i < S.length; i++) {
        if(arr[i] === "A" || arr[i] === "D") {
          if(arr[i] === arr[i+1]) {
            count+=1;
        }
        else{
          B.push(count);
          count = 1;
        }
      }
    }
      return B;
    };

Count the Heads

Tina is given a string S which contains the first letter of all the student names in her class. She got a curiosity to check how many people have their names starting from the same alphabet. So given a string S , she decided to write a code that finds out the count of characters that occur more than once in the string.

    var Count_Occ = (s) => {
      
      const alphabets = [];
      for(let i = 0; i < s.length; i++) {
        alphabets.push(s.charAt(i));
      }
      
      const repeatedValue = alphabets.filter( (alphabet) =>alphabets.indexOf(alphabet) !== alphabets.lastIndexOf(alphabet) )
      repeatedValue.sort();
      let result = ''
      
      let count = 1;
      for(let i = 0; i < repeatedValue.length; i++) {
        if(repeatedValue[i] === repeatedValue[i+1]) {
              count+=1;
      }
      else{
        result += repeatedValue[i] + count;
        count = 1;
      }
    }
      return result;
    };

Find the Winner

You are given a string S consisting of two letters A and D, where each character represent the winner of N games played between Aditya and Danish, where letter A represents the win of Aditya and letter D represents the win of Danish. You need to find out the that which player wins the maximum number of games or there is a draw between them.

var Game_Winner = (S) => {
  
  let countA = 0; 
  let countD = 0;
  for(let character of S) {
    if(character === "A") {
      countA+=1;
    }
    else if(character === "D") {
      countD+=1;
    }
    
  }
    
    if(countA > countD) {
      return "Aditya";
    }
    
    else if (countD > countA) {
      return "Danish";
    }
    
    else {
      return "Draw";
    }
    
 };

Plaindrome Check

You are given a string S, Your task is to check wether the given string is a Plaindrome or not.

A Plaindrome is a string, which turnout same when read in reverse direction.

Example: "naman" is a Palindrome.

var Palin_Check = (str) => {
  
  let isPalindrome = false;
	for(let i = 0; i < str.length; i++) {
	  if(str.charAt(i) === str.charAt(str.length-1-i)) {
	        isPalindrome = true;
	  }
	  
	  else {
	    isPalindrome = false;
	  }
	}
	
	if(isPalindrome === true) {
	  return "True";
	}
	
	else {
	  return "False";
	}
};
