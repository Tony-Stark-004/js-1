Find Sum

You are given an integer N, and your task is to find the sum of all the even integers starting from 1 upto N (N inclusive).

const findSum = (n) => {
  
        var sum = 0;
        while(n !== 0) {
          
          if(n%2 === 0) {

          }
        
              n = n-1;
